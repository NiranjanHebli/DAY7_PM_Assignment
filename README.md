## Overview

This project contains Python scripts that can be executed to perform specific tasks which were mentioned in the assignment.

## Prerequisites

- Python 3.x installed on your system

## Deliverables

- `finance_calculator.py` — interactive personal-finance comparison tool (Indian currency formatting, health scoring).
- `type_analyser.py` — simple utility that analyzes Python values and prints their type, truthiness, and length.

## Usage

- Run the finance tool (interactive):

```bash
python3 finance_calculator.py
```

- Run the type analyser (non-interactive):

```bash
python3 type_analyser.py
```
